# Smart Pantry

A production-ready pantry management system developed with Spring Boot.

## Features

- User Authentication
- Household Management
- Pantry Inventory
- Expiration Tracking
- Shopping List
- Notifications
- Consumption Analytics
- AI Recipe Recommendation

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- PostgreSQL
- Docker
- JWT
- Hibernate
- Flyway
- Swagger
